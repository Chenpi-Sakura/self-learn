# demo-serif

Notion-style × UKIYO Serif typography demo. A standalone Vite + React + TypeScript project that runs alongside `demo/` on port 5174.

## Quick Start

```bash
npm install
npm run dev
```

The dev server will start on port 5174 (port 5173 is reserved for `demo/`).

## Font Sources

Serif typefaces are copied (not generated) from `D:\Projects\SelfLearn\fonts\`:

- `FlyFlowerSong-OTF.otf`
- `FlyFlowerSong-TTF.ttf`
- `HedvigLettersSerif-Regular.otf`
- `HedvigLettersSerif-Regular.ttf`

They live in `public/fonts/` and are served as static assets.
